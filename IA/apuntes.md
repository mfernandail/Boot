# LLM - Versión Apuntes de Estudio

## Definición Rápida

Un LLM es una IA que aprende leyendo mucho texto y puede generar
respuestas parecidas a las humanas.

---

## Ideas Clave

- Aprende prediciendo la siguiente palabra.
- Usa millones o miles de millones de parámetros.
- Está basado en la arquitectura Transformer.
- Tiene memoria limitada (context window).

---

## Fases

### Pre-entrenamiento

Aprende patrones generales del lenguaje.

### Fine-tuning

Se especializa en tareas específicas.

### Inferencia

Es cuando el usuario lo usa.

---

## Parámetros

- Son los valores internos del modelo.
- Más parámetros → más capacidad.
- Más parámetros → más recursos necesarios.

---

## Context Window

- Es la memoria temporal del modelo.
- Se mide en tokens.
- Más grande = mejor comprensión de textos largos.

---

## System Prompt

- Configuración inicial.
- Define cómo debe comportarse el modelo.

---

## Prompt Message

- Es lo que escribe el usuario.
- Es la base de la respuesta generada.

---

## Resumen Express

LLM = Modelo grande + muchos datos + predicción de palabras + memoria
limitada + instrucciones + pregunta del usuario.
